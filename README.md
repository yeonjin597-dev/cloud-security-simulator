# 🛡️ 보안 사고 대응 실습 기반 퀴즈

이 프로젝트는 실전형 클라우드 보안 사고 대응 능력을 기르기 위한 시뮬레이션 웹 애플리케이션입니다!

## 👤 개발자 정보
- **학번:** 2025404018
- **성명:** 정연진

## 🚀 주요 기능
- **로그인 시스템:** `st.session_state`를 활용한 사용자 인증 및 접근 제어
- **성능 최적화:** `@st.cache_data`를 적용한 대용량 보안 로그 캐싱 처리
- **보안 대응 퀴즈:** 6단계 실무 사고 대응 시나리오 및 상세 기술 해설
- **리포팅 시스템:** 서버 HP 잔량에 따른 실시간 대응 등급 산출

## 🛠️ 기술 스택
- Python, Streamlit, Pandas
